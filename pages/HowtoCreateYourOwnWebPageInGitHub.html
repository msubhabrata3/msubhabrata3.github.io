How to create your own web page In GitHub

  1. Create a new account in https://github.com/
  2. Create a new repository as <yourusername>.github.io
  3. Use any HTML editor to create index.html file as follows
  4. Find a profile picture of you and rename it to bioPic.jpeg
  5. View index.html in a browser and adjust the width and height accordingly
  6. Upload index.html and bioPic.jpeg files to the same Github repository
  7. Make sure both files are uploaded to the same folder
  8. Open any browser and go to https://<yourusername>.github.io/
